# svaf-draw

> 🎨 /draw 前端 — AI 绘图助手聊天界面

SvelteKit 前端，与 [entaiaki/local-draw](https://github.com/entaiaki/local-draw) 配套使用的 AI 绘图助手界面。

基于 [afoim/svaf](https://github.com/afoim/svaf) 稀疏检出。

## 本地开发

```bash
pnpm install
pnpm dev           # :5173, API 自动代理到 :8080
```

## 功能
- AI 绘图助手聊天（自然语言→生图）
- 自动批准 / 跟踪模式
- 生图画廊（我的/精选）
- 角色/画风选择
- 多工作流切换 + 分辨率选择
- 图生图 (img2img)
